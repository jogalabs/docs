    Javascript

    Tentamos ao máximo, deixar nosso framework sem dependências de Javascript. Como você já viu, até nossos tooltips são independentes. Fizemos esse esforço para que suas aplicações web fiquem o mais leve possível.

    Mas nem tudo é possível sem o uso dele. Nessa seção você vai encontrar não apenas complementos para nosso framework como também snippets úteis no dia-a-dia. E sabe porque? Porque eu não aguento mais reescrever códigos. E achamos que você também. Por isso, estamos adotando o conceito DRY aqui. Tudo o que nos é útil será disponibilizado para vocês também.

    Além disso, usamos vários scripts de terceiros e mostramos como fizemos a integração deles com nosso framework.
