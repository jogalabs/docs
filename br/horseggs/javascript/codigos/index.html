    • Códigos

    Deseja compartilhar seus conhecimentos ninja com o mundo? Preparamos uma formatação bem legal para você mostrar sua arte de programar, baseados nos temas Solarized do Ethan Schoonover e Monokai de Wimer Hazenberg. É isso mesmo! Você tem duas opções de temas pra mostrar toda sua habilidade usando o Horseggs: Um tema claro e outro escuro. Use o que melhor se encaixar no seu estilo.

    Esse recurso é dependente do Google Code Prettify, portanto não se esqueça de incluí-lo (conforme abaixo) em todas as páginas em que desejar um código mais belo. 
    <script src="/assets/script/libs/prettify/r298/prettify.js"></script>

    Como funciona?

    A documentação deste egg acaba se mesclando com a documentação original do Google Code Prettify, já que esse script é bem prático. Assim, você aprende tudo em um lugar só. ;)

    Como mostrado acima, basta acrescentar class="prettyprint" às tags <code> ou <pre>. O script busca por essa combinação (tag + classe) e adiciona várias tags <span> com cores diferentes lá dentro, de acordo com a linguagem de programação utilizada.

    Temas

    Aqui nessa documentação, por exemplo, você verá o tema Solarized aplicado por toda parte. Para usá-lo, você não precisa fazer nada além do básico - já que esse é o nosso estilo padrão. Basta adicionar class="prettyprint".

    <pre>
    <html>
        <head>
            <title>Example</title>
            <link rel="stylesheet" type="text/css" href="horseggs.css" />
        </head>
        <body>
            <div class="style">Just some random test.</div>
        </body>
    </html>
</pre>

    Para usar o estilo Monokai (escuro), é fácil. Basta adicionar class="prettyprint dark" que seu código ficará assim:

    <pre class="prettyprint dark">
    <html>
        <head>
            <title>Example</title>
            <link rel="stylesheet" type="text/css" href="horseggs.css" />
        </head>
        <body>
            <div class="style">Just some random test.</div>
        </body>
    </html>
</pre>

    Linhas

    No exemplo acima, você notou como mostramos cada linha, com a cor de background intercalado e numeração? Fica bem legal, né? Só que isso não acontece por padrão. Para adicioná-las, aplique também a classe class="prettyprint linenums". Veja a diferença abaixo:

    Line 1
Line 2
Line 3
Line 4

    Line 1
Line 2
Line 3
Line 4

    E se por exemplo, você estiver citando apenas uma parte do código, e a primeira linha não for a linha 1? Não tem problema. Basta indicar, na própria classe, qual é a primeira linha, assim: .

    Line 4
Line 5
Line 6
Line 7

    Linguagens Suportadas

    "De fábrica", o script suporta as seguintes linguagens:

    • bsh
    • c
    • cc
    • cpp
    • cs
    • csh
    • cyc
    • cv
    • htm
    • html
    • java
    • js
    • m
    • mxml
    • perl
    • pl
    • pm
    • py
    • rb
    • sh
    • xhtml
    • xml
    • xsl

    Nem sempre o Google Code Prettify irá acertar a linguagem de programação utilizada nas suas tags. E nesse caso, você pode dar uma dica para o script, acrescentando class="lang-" com o nome da linguagem na tag.

    <blockquote class="right">...</blockquote>

    Domina outra linguagem? Nesse caso, você pode usar algumas extensões. Por exemplo, aqui nessa documentação, uso a extensão de CSS.

    <script src="/assets/script/libs/prettify/lang-css.js"></script>

    Para mais informações sobre as linguagens suportadas, acesse a documentação original.
