    Scroll 2

    <script src="assets/script/libs/horseggs/egg.scroll2.min.js"></script>

    Com essa simples função você pode deslizar o scroll do usuário automaticamente até uma determinada posição, como o topo da tela ou âncoras, espalhadas pelo conteúdo.

    Deslizar até um elemento

    <script type="text/javascript">
    $.egg.scroll2('#scroll-2-top');
</script>

    Basta passar o ID do elemento desejado para levar seu usuário até ele. Vamos testar? Clique no botão abaixo para deslizar até nosso segundo tópico.

    E tem mais uma coisa...

    Observe o layout da nossa documentação. Você notou que temos um menu fixo, que acompanha o scroll e fica por cima do conteúdo, certo? Para compensar isso na hora do scroll, você pode passar um valor de recuo (em pixels) pra função. Dessa forma você pode elimina a altura de qualquer elemento do seu layout que possa "confundir" o scroll.

    <script type="text/javascript">
    $.egg.scroll2('#topic', 192);
</script>
