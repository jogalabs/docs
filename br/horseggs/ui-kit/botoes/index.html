    Estilo flat.

    Estrutura

    Basta adicionar class="button" ao elemento desejado para tranformá-lo em um lindo botão flat responsivo.

    <button type="button" class="button">Button</button>
<input type="button" value="Button" class="button">
<a href="#" class="button">Button</button>
    Button

    Cores

    Você deve ter notado que a cor do botão acima não está das melhores, certo? Isso é proposital, já que nós sabemos que você vai querer escolher uma cor bem legal pra ele - sem contar que assim evitamos a duplicação desnecessária de estilos, certo? Disponibilizamos diversas cores pré-definidas para você usar. Para isso, basta adicionar o nome da cor junto à seu <input="button">. Vamos lá, clique e confira o efeito elegante que adicionamos.

    <button type="button" class="button blue">Button</button>
<button type="button" class="button cyan">Button</button>
<button type="button" class="button green">Button</button>
<button type="button" class="button brown">Button</button>

    Note que o último exemplo, removemos o estilo do botão para fazer com que o botão simule um link. Esse recurso é bem interessante, quando precisamos dar pesos diferentes a botões que estão um do lado do outro.

    Cores personalizadas Mixin

    Achamos que essas cores são o suficiente para atender a maioria dos casos, como botões de sucesso e erro, por exemplo. Mas digamos que você queira criar uma nova cor de botão, porque seu chefe exige que ele tenha a cor do logo da empresa. Não tem problema! Em sua folha de estilo .LESS, adicione o seguinte código, alterando .color-name por um nome de referência dessa cor e pelo código hexadecimal da mesma (Ex: #F0F0F0).

    button.color-name { @include flat-button-color (#); }

    Botões agrupados

    Para agrupar seus botões é fácil. Basta agrupá-los no seu markup em um elemento com class="horizontal-stack".

    <div class="horizontal-stack">
    <button type="button" class="button blue">Button 1</button>
    <button type="button" class="button blue">Button 2</button>
    <button type="button" class="button blue">Button 3</button>
</div>

    Precisa destes botões na vertical? Não tem problema. Basta mudar a classe do grupo para... É isso aí, você adivinhou!

    <div class="vertical-stack">
    <button type="button" class="button blue">Button 1</button>
    <button type="button" class="button blue">Button 2</button>
    <button type="button" class="button blue">Button 3</button>
</div>

    Quer deixar os botões da borda mais arredondados? É pra já!

    <div class="horizontal-stack rounded">
    <button type="button" class="button blue moderate">Button 1</button>
    <button type="button" class="button blue moderate">Button 2</button>
    <button type="button" class="button blue moderate">Button 3</button>
</div>

    Você notou no exemplo acima que o tamanho dos botões funcionou exatamente como você esperava? ;)

    Botão desabilitado

    Um erro comum que dá bastante dor de cabeça aos desenvolvedores acontece quando o usuário, por não ter nenhum feedback de que sua ação está sendo processada, clica duas, três vezes no botão Submit, gerando duplicações no banco de dados - e por consequência, erros. Adicione a propriedade disabled ou readonly após o primeiro clique para prevenir a entrada de dados pelo usuário e mudar ligeiramente o visual do botão. Ah, e um modal com a mensagem "processando" também ajuda. ;)

    <input type="submit" class="button yellow" disabled />

    Botões com ícones Mixin

    Que tal deixar seus botões ainda mais interessantes, acrescentando ícones? Usamos a Entypo Pictogram Suite, você pode adiciona qualquer um dos ícones disponíveis na sua biblioteca para personalizar seus botões.

    Por limitações da tag <input>, este recurso só está disponível para uso nas tags <a> ou <button>.

    Crie novos botões com ícones usando nosso Mixin em sua folha de estilo SASS. Tá vendo o código abaixo? É só substituir pelo código de quatro dígitos presente na biblioteca Entypo (aquele depois de "U+").

    .button.sample { @include button-icon(""); }        // Para botões comuns
.button.sample { @include button-social-icon(""); } // Para botões sociais

<button type="button" class="button with-icon red pinterest">Pinterest</button>
