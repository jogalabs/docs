    Estrutura

    Para garantir um layout realmente responsivo, nada melhor do que trabalhar com ícones de fontes. Qualquer tamanho, qualquer cor, qualquer estilo CSS. Com apenas uma simples <span>, você acrescenta qualquer um dos ícones abaixo no seu projeto.

    Começar é simples. Simplesmente, inclua a classe class="icon" seguido do símbolo que você deseja utilizar na sua página. Fácil, não?

    Ícones customizados Mixin

    Imagino que a maioria das suas necessidades serão supridas com os ícones acima, certo? Porém, alguns projetos exigem uma dose maior de personalização. Para isso, você pode usar o Mixin abaixo:

    @include load-icon-font ('CLASSE','NOME','CAMINHO');

    O que este mixin faz? Primeiro, ele carrega uma fonte de ícones com o NOME acima que esteja no CAMINHO passado. Depois, ele cria uma CLASSE com a fonte já atribuída a ela. A ideia é que você use essa classe com todos os ícones que você deseja usar a mesma fonte. Exatamente como fizemos acima, com class="icon". Por exemplo:

    @include load-icon-font ('.padrao','entypo','/assets/typo/');

    Note que em CAMINHO você deve passar apenas o caminho até a pasta onde se encontram os arquivos. Lá dentro, o nome do arquivo deve ser exatamente igual ao NOME da fonte.
