    Breadcrumb

    Como é ruim se sentir perdido, não? É por isso que usamos Breadcrumbs. Eles indicam a localização da página atual dentro da estrutura do site. Montar um breadcrumb é muito fácil. Basta criar uma lista com class="breadcrumb", acrescentando um link em cada linha. O último item da lista, entretanto, não deve ser um link - já que o usuário já se encontra nesta página.

    <ul class="breadcrumb">
    <li ><a href="#1" title="Página 1">Página 1</a></li>
    <li ><a href="#2" title="Página 2">Página 2</a></li>
    <li>Página 3</li>
</ul>

    • Página 2
    • Página 3

    Na maioria das vezes o primeiro item do seu breadcrumb será a Home do site, certo? Essa é uma boa hora para trocar esse item por um ícone, deixando sua breadcrumb mais enxuta e elegante. Basta acrescentar class="home" no primeiro <li>

    <ul class="breadcrumb">
    <li class="home"><a href="#home" title="Home"></a></li>
    <li><a href="#2" title="Página 2"></a>Página 2</li>
    <li>Página 3</li>
</ul>
    • Página 2
    • Página 3

    Como você percebeu, montar um breadcrumb é muito fácil. Ele automaticamente irá preencher 100% do container que o contém. E em dispositivos mobile, cada link ocupa uma linha. Mude o tamanho desta janela e confira.

    Tabs

    Quer usar tabs para mudar o conteúdo? Fácil. Basta envolver os botões de troca de conteúdo (<a> ou <button>) em um elemento com class="nav tabs".

    <div class="tabs">
    <button type="button" class="ON">Conteúdo 1</button>
    <button type="button">Conteúdo 2</button>
    <a href="#">Conteúdo 3</a>
</div>

    Paginação

    Quantas páginas! Vamos organizar isso? Insira uma lista de paginação no rodapé - ou no cabeçalho, você manda. Para isso use uma lista com class="pagination". O elemento selecionado deve ser um <span>. O restante, <a> ou <button>.

    <ul class="pagination">
    <li ><a href="#">«</a></li>
    <li ><a href="#1">1</a></li>
    <li ><a href="#2">2</a></li>
    <li>3</li>
    <li ><a href="#4">4</a></li>
    <li ><a href="#5">5</a></li>
    <li ><a href="#6">6</a></li>
    <li ><a href="#7">7</a></li>
    <li ><a href="#">»</a></li>
</ul>
