Alertas, Rótulos e Bolhas

Contexturalização através das cores.

Alertas

Envolva qualquer texto em um elemento usando class="alert " para transformá-lo em um alerta. Por exemplo:

<p class="alert error">Este é um alerta de erro.</p>

Temos 7 contextos disponíveis para você usar em seus projetos. Confira abaixo:

Este é um alerta de sucesso. Ative usando class="alert success".

Este é um alerta de informação. Ative usando class="alert info".

Este é um alerta com uma nota. Ative usando class="alert notice".

Este é um alerta pra dizer algo importante. Ative usando class="alert important".

Este é um alerta pra chamar atenção. Ative usando class="alert warning".

Este é um alerta de erro. Ative usando class="alert error".

Este é um alerta de perigo. Ative usando class="alert danger".

Não existe razão no universo expandido de Star Wars para você incluir um alerta sem um contexto (como sucesso ou erro, por exemplo). Por isso, se você usar apenas class="alert", nada irá acontecer.

Alertas com ícones

Além da cor, incluímos alguns ícones para deixar a mensagem do seu alerta ainda mais clara. Para incluir, basta acrescentar class="with-icon" ao elemento que está recebendo o alerta.

Alertas dispensáveis

Você pode dar a opção para o usuário fechar o alerta a qualquer momento. Basta acrescentar class="dismissable" ao alerta para incluir um botão fechar.

Etiquetas e Bolhas

Etiquetas são úteis para destacar algum texto, em títulos ou no decorrer de um texto. Por exemplo, para destacar novidades ou que algo foi atualizado.

  • Confira este post!NOVO

<span class="label important">NOVO</span>

E assim como os alertas, eles também estão disponíveis em diversas cores. Basta acrescentar a classe referente a cada cor, conforme visto nos alertas (logo acima):

default success info notice important warning error danger

Já as bolhas servem para destacar números. Por exemplo, itens não lidos, número de novidades ou atualizações. Veja um exemplo:

  • Caixa de Entrada 49
<h5>Caixa de Entrada <span class="badge danger">49</span></h5>

Preciso dizer que as cores estão todas disponíveis?

1 2 3 4 5 6 7 8

Se você quiser, pode jogar o badge para o final do elemento que contém o badge. Acrescente class="right" e pronto!

Atualizações 37

As bolhas, inclusive, possuem uma "cor" a mais. Quando o elemento que o contém possuir alguma cor, você pode usar class="negative" para que a bolha ganhe um fundo branco. Dessa forma apenas o texto terá alguma cor de contexto.
