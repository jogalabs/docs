    Tooltips

    Quer um jeito rápido e fácil de apresentar mais informações sobre uma ação ou termo da sua página. Tudo isso, sem ocupar espaço no layout e com uma markup bem simples.

    O link a seguir é um <a href="#" class="has-tip top" title="Tooltip">tooltip</a>.
    O link a seguir possui uma dica.
     

    É só adicionar class="is-tip" ao elemento que dará a dica, que ele irá automaticamente criar a dica usando seu title="". Legal, não? Opa! Deve ter algo errado. Essa seção era pra estar dentro de "Javascripts", certo? Errado! Esse recurso usa apenas CSS. E mais: Nenhuma imagem.

    Todo browser já possui um comportamento padrão para exibição dos titles, e isso acaba duplicando a informação na página. Quer acabar com esse problema? Visite a seção Javascripts e aplique nosso snippet.

    Cores

    É isso mesmo: dicas coloridas. Porque além de informção, as dicas podem servir para mais!

    O link a seguir é um <a href="#" class="has-tip top-left info" title="Essa é uma dica colorida!">dica colorida</a>.
    O link a seguir tem uma dica colorida.

    Cores Personalizadas Mixin

    Assim como com os botões, você também pode criar dicas com cores personalizadas. É super fácil. Em sua folha de estilo .LESS, adicione o seguinte código, alterando .color-name por um nome de referência dessa cor e pelo código hexadecimal da mesma (Ex: #F0F0F0).

    .has-tip.color-name { .tip-color (#); }

    Formulários

    Dicas também podem ser úteis em formulários, tanto para apresentar exemplos como para destacar erros (ou acertos) de validação. Você pode, por exemplo, dar uma dica de como preencher corretamente aquele campo errado, ou apresentar um visto no campo que foi checado via Ajax.

    Para usar, não tem segredos. Basta, basicamente, seguir o que já ensinamos até aqui. Selecione o campo de texto e veja como funciona:

    <div class="input group">
<input type="text" class="standard error">
<span class="is-tip top-right error">Mensagem da Dica</span>
</div>
    Mensagem da Dica
     

    Mostramos a dica quando o usuário seleciona o campo respectivo. Simples, não? Para usá-lo, basta seguir o markup acima: Aplique class="is-tip" ao elemento que dará a dica (geralmente, um <span>) e em seguida, envolva o campo de texto e a dica com um <div class="input group" />. Você tem todas as posições e cores à sua disposição.

    Aliás, envolva sua dica com qualquer elemento com class="group" que ela aparecerá quando você passar o mouse sobre esse elemento. ;)

    E tem mais uma coisa...

    Prefere deixar sua dica sempre visível? Basta acrescentar class="always-visible".

    O link a seguir possui uma <a href="#" class="has-tip top-right always-visible" title="Dica sempre visível!">dica visível</a>.
    O link a seguir possui uma dica visível.
